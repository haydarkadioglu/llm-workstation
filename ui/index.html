Exposed MCP Tools

get_telemetry

Retrieves real-time used/total VRAM status, active GPU names, model loading states, and saniyede üretilen tokens metrics.

load_model

Takes a `model_id` string arguments payload to dynamically trigger background downloads and hot-swaps of HF models.

unload_model

Instantly drops the model from memory and purges GPU CUDA caches to free VRAM for other server tasks.

generate

Generates text from the loaded model. Supports multimodal inputs via `image_base64` parameter if a vision model is loaded.

Workstation Diagnostics

This workstation is configured to automatically download the Cloudflare Tunnel binaries on start, meaning it is fully portable. Quantization defaults use 4-bit Double Quantization to squeeze larger instruct models (up to 8B/13B parameters) into consumer or Colab T4 GPU VRAM limits.
